(기본적인 숙제를 하시면서 스펙업을 조금씩 하시면 클럽 또는 오래 하신 유저분들에게 물어보시면서 상위 컨텐츠 진입 하시면 됩니다.)

    • 렐릭 / 룬의던전 / 과제 / 크라운(출석체크) / 클럽던전
    • 머큐리얼케이브 보스 + 루미너스(머루) / 프시키 허상 / 프시키 미궁
    • 힘의근원 / 1인 프라바방어전
    • 멜카르트DPS / 발굴지
    • 심연의보물창고

  • [주간]

    • 카타콤 헬(카헬) - 어비스 심층 보스전 일반(어일) - 신조의둥지 일반(신일)
    • 어밴던로드
